# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [0.1.0] - 2026-03-17

### Added

- Button component with `primary`, `secondary`, `destructive`, and `ghost` variants
- Card component with `CardHeader`, `CardContent`, and `CardFooter` subcomponents
- Input component with label, helper text, and error state support
- Modal component with overlay and focus management
- Tabs component with `TabsList`, `TabsTrigger`, and `TabsContent`
- `cn` utility for merging Tailwind classes
- Theme CSS with semantic design tokens (light and dark mode)
- Storybook setup with stories for all components
- Vitest setup with Button test suite
